## 🌸 项目简介

这是一个**30天具身智能速成学习日历**，专为想要快速入门具身智能（Embodied AI）的研究者和学习者设计。

每天三个时间段（早/午/晚）安排具体任务，从工程基础到前沿实战，循序渐进地带你掌握具身智能的核心技术栈。

***学习笔记参考*** [https://github.com/Feijiake/EmbodyAI-30day-Notebook](https://github.com/Feijiake/EmbodyAI-30day-Notebook)

## ✨ 功能特色

- 🌈 **交互式日历** - 鼠标悬停查看每天任务概览
- 📝 **任务追踪** - 每日三个时段任务，点击完成记录
- 💾 **进度保存** - LocalStorage自动保存学习进度
- 💌 **温柔鼓励** - 每天配送阿俊的心灵鸡汤
- 🎀 **可爱特效** - 完成任务时有爱心爆炸动画
- 🌸 **优雅视觉** - Apple风格粉嫩配色，赏心悦目

---

## 🚀 使用方法

**Step 1：** 直接在浏览器中打开 `UI.html`

**Step 2：** 选择任意一天开始学习，点击任务复选框标记完成

**Step 3：** 完成后日历格子会变成粉色♥，自动跳转到下一天

**Step 4：** 想要重新开始？点击右上角「重新开始吧」即可
